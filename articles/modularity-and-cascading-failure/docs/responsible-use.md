# Responsible Use

This scaffold is not a safety certification or emergency-management command tool.

Do not use it to:

- claim a system is safe without empirical dependency audits and stress testing
- rank communities, agencies, hospitals, ecosystems, or infrastructure providers without context
- justify service withdrawal, local abandonment, or underinvestment
- hide uncertainty or contested assumptions behind model output
- ignore common-mode failure
- treat justice-weighted impact as optional
- replace engineering review, ecological monitoring, public-health expertise, local knowledge, or accountable governance

Use it to make cascade assumptions visible, compare containment strategies transparently, detect hidden dependencies, and support better resilience planning.
