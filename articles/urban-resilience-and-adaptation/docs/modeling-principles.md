# Modeling Principles

This companion scaffold models urban resilience as a social-ecological-infrastructural and governance problem.

## Core variables

- exposure reduction
- vulnerability reduction
- service continuity
- adaptive capacity
- ecological buffering
- equity protection
- maladaptation risk
- hazard intensity
- infrastructure disruption
- health burden
- housing stress
- market stress
- community capacity
- dependency amplification
- system function under disturbance

## Core modeling logic

Urban resilience value rises when exposure reduction, vulnerability reduction, service continuity, adaptive capacity, ecological buffering, and equity protection increase. Resilience value falls when maladaptation risk, hazard intensity, dependency amplification, infrastructure disruption, housing stress, health burden, market stress, and unequal burden increase.

The scaffold calculates:

1. urban strategy value under multiple priority scenarios
2. equity-adjusted resilience value
3. urban system function under repeated stress
4. maladaptation-risk diagnostics
5. service-continuity, ecological-buffering, and equity review triggers
6. Monte Carlo robustness under uncertainty
7. synthetic classifier outputs for high-value, low-maladaptation urban strategies

## Interpretation

These examples are transparent modeling scaffolds, not empirical public-safety claims. Real urban resilience analysis requires local hazard data, climate projections, infrastructure records, housing data, public-health evidence, ecological monitoring, service-continuity data, community participation, anti-displacement review, and accountable governance.
