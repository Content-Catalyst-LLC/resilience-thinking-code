# Modeling Principles

This companion scaffold models community resilience as a social-capital, institutional-capacity, infrastructure-access, economic-security, information-quality, adaptive-capacity, and equity problem.

## Core variables

- social capital
- institutional capacity
- infrastructure access
- economic security
- information quality
- adaptive capacity
- equity protection
- implementation burden
- hazard intensity
- service disruption
- institutional pressure
- communication stress
- economic shock
- care burden
- equity burden
- dependency amplification
- community function under disturbance

## Core modeling logic

Community resilience value rises when social capital, institutional capacity, infrastructure access, economic security, information quality, adaptive capacity, and equity protection increase. Resilience value falls when implementation burden, hazard intensity, service disruption, institutional pressure, communication stress, economic shock, care burden, equity burden, and dependency amplification increase.

The scaffold calculates strategy value, equity-adjusted resilience value, community function under repeated stress, implementation-burden diagnostics, and Monte Carlo robustness under uncertainty.
