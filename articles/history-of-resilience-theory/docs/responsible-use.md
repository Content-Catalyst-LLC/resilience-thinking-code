# Responsible Use

Historical modeling can clarify intellectual development, but it can also oversimplify.

Use this scaffold to:

- organize historical phases
- map concept emergence
- compare conceptual expansion across periods
- produce transparent educational visualizations
- connect narrative history to reproducible workflows

Do not use this scaffold to:

- claim definitive bibliometric measurement from synthetic data
- erase disciplinary disagreement
- flatten critical debates into a single score
- replace careful historical reading
- treat "influence scores" as objective truth

The model is a tool for interpretation, not a substitute for interpretation.
