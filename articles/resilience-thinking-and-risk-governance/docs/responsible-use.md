# Responsible Use

This scaffold is not an automatic risk-governance decision system.

Do not use it to:

- rank communities without participatory review
- justify disinvestment in places labeled high risk
- replace technical risk assessment or domain expertise
- hide uncertainty behind synthetic scores
- treat governance legitimacy as a purely technical metric
- ignore justice, accountability, and public responsibility

Use it to support transparent inquiry into how hazard, exposure, vulnerability, capacity, governance legitimacy, and adaptive capacity shape resilience outcomes.
