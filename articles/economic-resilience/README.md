# Economic Resilience

Companion repository folder for the article **Economic Resilience**.

This directory provides reproducible workflows for economic resilience strategy comparison, resistance and recovery analysis, adaptation and transformation diagnostics, equity and institutional-capacity review, implementation-burden analysis, regional economic function simulation, structural fragility review, and uncertainty-aware economic resilience planning.

## Correct article directory

```text
articles/economic-resilience/
```

## Purpose

This scaffold supports a professional economic resilience modeling workflow:

1. Define economic resilience strategies across resistance, recovery, adaptability, transformability, equity protection, institutional capacity, and implementation burden.
2. Compare strategies under balanced, resistance-first, recovery-first, adaptation-first, transformation-first, equity-first, institution-first, and implementation-aware priorities.
3. Simulate economic function under financial stress, supply-chain disruption, climate shock, labor displacement, fiscal stress, inflation pressure, insurance retreat, and compound economic disruption.
4. Estimate equity-adjusted, institution-adjusted, and implementation-adjusted economic resilience value.
5. Diagnose resistance gaps, recovery gaps, adaptation gaps, transformation pathway gaps, equity-safeguard gaps, institutional-capacity gaps, and implementation-burden triggers.
6. Model uncertainty using Monte Carlo simulation.
7. Support responsible interpretation across economic resilience, regional resilience, supply-chain resilience, financial system resilience, labor-market adaptation, climate transition, public finance, infrastructure, social vulnerability, and just transformation.
8. Provide multi-language examples in Python, R, Julia, SQL, Rust, Go, C, C++, and Fortran.
9. Export reproducible tables, figures, documentation, and notebook placeholders.

## Responsible-use note

The data are synthetic and the models are methodological examples. They are intended for learning, reproducible article companions, dashboard prototyping, resilience strategy comparison, economic systems modeling, and professional workflow design. Real economic resilience analysis requires local data, distributional analysis, public finance review, labor-market evidence, institutional history, social vulnerability analysis, climate risk assessment, financial-sector data, and affected-community participation.

Do not use these examples to automate public economic decisions, rank regions without context, justify austerity or abandonment, hide unequal recovery behind aggregate scores, replace democratic planning, or treat economic resilience as GDP recovery alone.

## Quick start

Run the dependency-light Python workflow:

```bash
python3 python/economic_resilience_standard.py
```

Optional advanced Python workflow:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements-advanced.txt
python python/economic_resilience_advanced.py
```

Run smoke checks:

```bash
bash smoke-test.sh
```
