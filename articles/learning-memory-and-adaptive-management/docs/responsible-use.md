# Responsible Use

This scaffold is not a universal adaptive-capacity scoring system.

Do not use it to:

- claim exact learning capacity without empirical evidence
- rank communities, agencies, ecosystems, or institutions without context
- justify experimentation on vulnerable groups without consent, safeguards, and accountability
- ignore community memory, worker knowledge, Indigenous knowledge, or histories of harm
- treat after-action reports as evidence of learning unless implementation follows
- hide uncertainty behind model output
- replace ecological monitoring, technical review, public participation, or accountable governance

Use it to make learning assumptions visible, compare adaptive-management strategies transparently, identify forgetting pressure, and connect monitoring to better decisions.
