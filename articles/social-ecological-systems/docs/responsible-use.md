# Responsible Use

This scaffold is not an automatic ranking system for communities, ecosystems, or resource systems.

Do not use it to:

- rank communities without participatory review
- claim precise thresholds from synthetic data
- replace ecological expertise or institutional analysis
- extract local or Indigenous knowledge without authority and consent
- justify disinvestment in systems labeled vulnerable
- hide structural injustice behind systems language
- treat resilience as automatically desirable persistence

Use it to support transparent inquiry into how ecological condition, governance, livelihoods, infrastructure, knowledge systems, market pressure, climate exposure, and adaptive capacity shape social-ecological resilience.
