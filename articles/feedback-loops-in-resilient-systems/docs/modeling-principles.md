# Modeling Principles

This companion scaffold models feedback loops in resilient systems.

## Core variables

- reinforcing gain
- balancing strength
- feedback delay
- target state
- disturbance shock
- adaptive response
- signal quality
- system memory
- justice visibility
- loop polarity
- overshoot
- oscillation
- policy resistance

## Core modeling logic

Feedback-related resilience risk rises when reinforcing gain, disturbance load, and delay exceed balancing strength, adaptive capacity, signal quality, system memory, and justice visibility.

The scaffold calculates reinforcing, balancing, combined, and delayed feedback trajectories; delay-sensitivity comparisons; feedback-risk scores; loop-polarity diagnostics; policy-resistance examples; adaptive feedback and learning effects; and synthetic feedback-risk classifier outputs.
