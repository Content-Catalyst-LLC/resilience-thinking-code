# Modeling Principles

This companion scaffold models institutional resilience as a legitimacy, capacity, flexibility, coordination, learning, accountability, and equity problem.

## Core variables

- legitimacy
- administrative capacity
- flexibility
- coordination
- learning
- accountability
- equity protection
- implementation burden
- shock intensity
- legitimacy pressure
- capacity burden
- coordination stress
- information stress
- legal/accountability stress
- equity burden
- dependency amplification
- institutional function under stress
- threshold risk

## Core modeling logic

Institutional resilience value rises when legitimacy, capacity, flexibility, coordination, learning, accountability, and equity protection increase. Resilience value falls when implementation burden, shock intensity, legitimacy pressure, capacity burden, coordination stress, information stress, legal/accountability stress, equity burden, and dependency amplification increase.

The scaffold calculates strategy value, equity-adjusted resilience value, institutional function under repeated stress, threshold-risk diagnostics, implementation-burden diagnostics, and Monte Carlo robustness under uncertainty.
