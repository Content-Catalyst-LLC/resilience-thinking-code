# Institutional Resilience

Companion repository folder for the article **Institutional Resilience**.

This directory provides reproducible workflows for institutional resilience strategy comparison, legitimacy and trust scoring, administrative-capacity diagnostics, coordination and learning review, equity-adjusted resilience value, implementation-burden review, institutional function simulation, threshold-risk review, and uncertainty-aware governance planning.

## Correct article directory

```text
articles/institutional-resilience/
```

## Purpose

This scaffold supports a professional institutional resilience modeling workflow:

1. Define strategies across legitimacy, administrative capacity, flexibility, coordination, learning, accountability, equity protection, and implementation burden.
2. Compare strategies under balanced, legitimacy-first, capacity-first, flexibility-first, coordination-first, learning-first, equity-first, and implementation-aware priorities.
3. Simulate institutional function under public trust shock, fiscal stress, cyber/data disruption, coordination failure, misinformation, legal conflict, workforce attrition, and compound governance stress.
4. Estimate equity-adjusted institutional resilience value and unequal access or administrative-burden risk.
5. Diagnose implementation-burden triggers, legitimacy gaps, administrative-capacity gaps, coordination failures, accountability weaknesses, learning failures, and threshold risk.
6. Model uncertainty using Monte Carlo simulation.
7. Support responsible interpretation across public agencies, local governance, regulatory institutions, courts, public-health institutions, utilities, civil society, and cross-scale governance systems.
8. Provide multi-language examples in Python, R, Julia, SQL, Rust, Go, C, C++, and Fortran.
9. Export reproducible tables, figures, documentation, and notebook placeholders.

## Responsible-use note

The data are synthetic and the models are methodological examples. They are intended for learning, reproducible article companions, dashboard prototyping, institutional diagnostics, and professional modeling design. Real institutional resilience analysis requires participatory governance, legal review, administrative data, trust and legitimacy research, equity audits, institutional-history analysis, public accountability, privacy safeguards, and affected-community participation.

Do not use these examples to rank institutions without context, automate public governance decisions, justify emergency overreach, certify institutional legitimacy, replace legal or administrative judgment, or hide unequal access and administrative burden behind a single resilience score.

## Quick start

Run the dependency-light Python workflow:

```bash
python3 python/institutional_resilience_standard.py
```

Optional advanced Python workflow:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements-advanced.txt
python python/institutional_resilience_advanced.py
```

Run smoke checks:

```bash
bash smoke-test.sh
```
