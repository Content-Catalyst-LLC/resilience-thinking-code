# Responsible Use

This scaffold is not a collapse-prediction system.

Do not use it to:

- claim exact regime-shift timing from synthetic data
- rank communities, ecosystems, institutions, or infrastructure systems without validated evidence
- justify abandonment, displacement, austerity, or coercive intervention
- treat warning signals as certainty
- ignore local knowledge, Indigenous knowledge, worker knowledge, resident reports, or frontline evidence
- replace accountable governance with model output
- hide uncertainty behind dashboards

Use it to support transparent inquiry into how regime shifts, early warning signals, adaptive capacity, memory, monitoring, and justice shape resilience before crisis.
