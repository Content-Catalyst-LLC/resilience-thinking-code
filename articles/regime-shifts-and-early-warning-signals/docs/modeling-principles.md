# Modeling Principles

This companion scaffold models regime shifts and early warning signals.

## Core variables

- system state
- external pressure
- reinforcing feedback strength
- rolling variance
- lag-1 autocorrelation
- recovery-speed proxy
- threshold-proximity score
- adaptive capacity
- system memory
- monitoring quality
- justice visibility
- regime state

## Core modeling logic

Regime-shift risk rises when pressure, reinforcing feedback, variability, and persistence increase while recovery speed, adaptive capacity, system memory, monitoring quality, and justice visibility decline.

The scaffold calculates:

1. nonlinear transition trajectories
2. lower-regime and upper-regime states
3. rolling variance
4. lag-1 autocorrelation
5. recovery-speed proxy
6. threshold-proximity score
7. synthetic regime-risk classifier outputs

## Interpretation

These examples are transparent modeling scaffolds, not empirical predictions. Early warning indicators must be interpreted with system history, causal knowledge, domain expertise, local knowledge, uncertainty review, and accountable governance.
