# Ecological Resilience and Ecosystem Stability

Companion article directory for **“Ecological Resilience and Ecosystem Stability.”**

## GitHub path

```text
articles/ecological-resilience-and-ecosystem-stability/
```

## Purpose

This companion scaffold supports reproducible exploration of the distinction between ecosystem stability and ecological resilience.

It includes:

- stability-versus-resilience profile scoring
- biodiversity, functional diversity, and response diversity indicators
- threshold-distance diagnostics
- ecological regime-shift simulation
- basin-width and resilience-margin modeling
- disturbance scenario comparison
- multi-language computational examples

## Quick start

```bash
python3 python/ecological_resilience_standard.py
bash smoke-test.sh
```

## Responsible use

The included data are synthetic. Real ecological resilience assessment requires calibrated field data, ecological expertise, long-term monitoring, local and Indigenous knowledge where appropriate, and accountable review.
