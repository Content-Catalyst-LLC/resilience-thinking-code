# Modeling Principles

This companion scaffold distinguishes ecosystem stability from ecological resilience.

## Stability variables

- short-run stability
- resistance
- recovery speed
- return toward a reference state
- local variability

## Ecological resilience variables

- biodiversity
- functional diversity
- response diversity
- threshold distance
- basin width
- regenerative capacity
- ecological memory
- connectivity
- disturbance exposure
- slow-variable pressure

## Core logic

Stability is modeled as short-run constancy or return. Ecological resilience is modeled as threshold-centered persistence: the capacity of an ecosystem to absorb disturbance without shifting into another regime.
