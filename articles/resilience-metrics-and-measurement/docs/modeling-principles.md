# Modeling Principles

This companion scaffold models resilience metrics and measurement frameworks.

## Core variables

- resistance coverage
- recovery insight
- adaptive-capacity visibility
- buffer visibility
- justice visibility
- data-quality transparency
- threshold blindness
- recovery time
- shock intensity
- recovery completeness
- affected population share
- metric robustness

## Core modeling logic

A resilience metric framework is more useful when it captures resistance, recovery, adaptive capacity, buffer capacity, justice visibility, data quality, and threshold sensitivity. It is weaker when it hides uncertainty, reduces resilience to current performance, ignores distribution, or remains blind to thresholds and feedback loops.

The scaffold calculates:

1. metric framework rankings under multiple priority scenarios
2. framework robustness under uncertainty
3. event-level resistance and recovery diagnostics
4. threshold-blindness penalties
5. synthetic classifier outputs for high-value measurement frameworks

## Interpretation

These examples are transparent modeling scaffolds, not empirical claims. Real resilience measurement requires system definition, relevant disturbance scenarios, critical function thresholds, domain expertise, community participation, and accountable governance.
