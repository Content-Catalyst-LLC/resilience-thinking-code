# Responsible Use

This scaffold is not an automatic resilience ranking system.

Do not use it to:

- rank communities without participatory review
- label systems as low resilience without addressing structural causes
- hide responsibility behind abstract systems language
- praise survival while ignoring underinvestment
- treat synthetic thresholds as real-world evidence
- replace engineering, ecological, public-health, or governance expertise

Use it to support transparent inquiry into how feedback, stocks, flows, delays, and boundaries shape resilience outcomes.
