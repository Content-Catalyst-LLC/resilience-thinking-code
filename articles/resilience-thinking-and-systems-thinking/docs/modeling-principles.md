# Modeling Principles

This companion scaffold connects systems thinking and resilience thinking.

## Systems-thinking variables

- feedback visibility
- boundary clarity
- leverage capacity
- delay management
- reinforcing vulnerability
- balancing repair
- feedback delay
- stocks and flows

## Resilience-thinking variables

- adaptive capacity
- redundancy
- threshold distance
- buffer capacity
- vulnerability pressure
- disturbance exposure
- resilience margin
- threshold-risk state

## Core modeling logic

Systems thinking explains how structure generates behavior. Resilience thinking asks whether that behavior preserves viability under disturbance.

The scaffold therefore simulates:

1. vulnerability stocks
2. reinforcing vulnerability feedback
3. balancing repair feedback
4. delayed response
5. disturbance load
6. adaptive response
7. resilience margin
8. threshold-risk flags

## Appropriate use

Use this scaffold for education, synthetic-data modeling, dashboard prototyping, article companion code, and professional methods design. Replace synthetic data with validated system-specific evidence before real-world use.
