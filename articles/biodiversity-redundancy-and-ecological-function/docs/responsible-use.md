# Responsible Use

This scaffold is not an automatic biodiversity ranking system.

Do not use it to:

- rank ecosystems, species, communities, or conservation priorities without field evidence
- claim precise ecological thresholds from synthetic data
- treat functional redundancy as evidence that species are replaceable
- ignore keystone species, rare species, cultural species, or intrinsic ecological value
- use biodiversity metrics to justify destruction through simplistic offsets
- extract local or Indigenous knowledge without authority and consent
- replace ecological expertise, governance review, or community accountability

Use it to support transparent inquiry into how biodiversity, redundancy, response diversity, ecological memory, connectivity, and disturbance shape ecological function.
