# Modeling Principles

This companion scaffold models biodiversity, redundancy, response diversity, and ecological function.

## Biodiversity-resilience variables

- species richness
- functional diversity
- functional redundancy
- response diversity
- connectivity
- ecological memory
- disturbance exposure
- trait contribution
- disturbance sensitivity
- recovery capacity
- functional output
- resilience margin

## Core modeling logic

Ecological function can remain visible while resilience declines. A function becomes fragile when species abundance, redundancy, response diversity, ecological memory, or connectivity are lost, especially under repeated or compound disturbance.

The scaffold calculates:

1. functional resilience profiles
2. redundancy and response-diversity diagnostics
3. species-trait function outputs
4. function-loss trajectories
5. disturbance scenario margins
6. threshold-risk flags

## Appropriate use

Use this scaffold for education, synthetic-data modeling, dashboard prototyping, article companion code, and professional methods design. Replace synthetic data with validated biodiversity, trait, genetic, habitat, and disturbance data before real-world use.
