# Validation Plan

A real maladaptive-resilience workflow should validate assumptions using ecological data, repeated-loss records, infrastructure condition data, public health indicators, social vulnerability evidence, housing data, labor conditions, climate scenarios, institutional records, budget and maintenance data, community knowledge, Indigenous and local knowledge where relevant, and after-action reviews.

Quantitative scoring should be reviewed with affected communities, domain experts, public agencies, ecologists, infrastructure operators, governance practitioners, social scientists, labor or housing advocates where relevant, and technical experts.
