# Responsible Use

The models in this folder are synthetic demonstrations. They are appropriate for:

- learning resilience concepts
- prototyping indicators
- comparing scenario assumptions
- testing reproducible workflows
- designing dashboards that will later be reviewed by domain experts

They are not appropriate for:

- automatic policy decisions
- ranking communities without context
- replacing participatory planning
- claiming prediction where only exploratory modeling exists
- obscuring responsibility for structural vulnerability

Resilience analysis should be paired with public accountability, ecological evidence, historical context, and affected-community participation.
