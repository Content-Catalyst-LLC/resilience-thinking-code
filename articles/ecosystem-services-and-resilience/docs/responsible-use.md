# Responsible Use

This scaffold is not an automatic ecosystem-service ranking system.

Do not use it to:

- rank communities or ecosystems without participatory review
- claim precise thresholds from synthetic data
- reduce ecosystem value to monetary service output
- justify ecological destruction through simplistic offsets
- ignore Indigenous rights, local knowledge, or cultural relationships to place
- treat access equity as a minor variable
- replace ecological expertise, governance review, or community accountability

Use it to support transparent inquiry into how ecological condition, functional diversity, redundancy, governance, disturbance, access, and resilience margins shape ecosystem-service futures.
