# Modeling Principles

This companion scaffold models public health system resilience as a prevention, detection, continuity, workforce, governance, trust, and equity problem.

## Core variables

- prevention
- detection
- service continuity
- workforce capacity
- adaptive governance
- trust
- equity protection
- implementation burden
- hazard intensity
- surge demand
- service disruption
- workforce burden
- trust pressure
- supply-chain stress
- equity burden
- dependency amplification
- system function under disturbance

## Core modeling logic

Public health resilience value rises when prevention, detection, service continuity, workforce capacity, adaptive governance, trust, and equity protection increase. Resilience value falls when implementation burden, hazard intensity, surge demand, service disruption, workforce burden, trust pressure, supply-chain stress, equity burden, and dependency amplification increase.

The scaffold calculates strategy value, equity-adjusted value, system function under repeated stress, implementation-burden diagnostics, and Monte Carlo robustness under uncertainty.
