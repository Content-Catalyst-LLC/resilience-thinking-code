# Slow Variables and Hidden System Change

Companion repository folder for the article **“Slow Variables and Hidden System Change.”**

This directory provides reproducible workflows for modeling slow-variable accumulation, threshold distance, hidden risk, adaptive capacity, ecological memory, institutional trust, maintenance backlog, climate pressure, fast-shock interaction, scenario comparison, and responsible slow-variable monitoring.

## Correct article directory

```text
articles/slow-variables-and-hidden-system-change/
```

## Purpose

This scaffold supports a professional slow-variable resilience workflow:

1. Represent slow variables as accumulated stocks that change through inflows, outflows, erosion, recovery, and investment.
2. Estimate hidden risk from maintenance backlog, climate pressure, trust decline, ecological memory loss, exposure, and feedback amplification.
3. Model threshold distance and the narrowing of system safety margins before visible crisis.
4. Simulate how fast shocks produce different outcomes depending on slow conditions already in place.
5. Compare neglect, moderate adaptation, and high-prevention scenarios.
6. Diagnose monitoring quality, signal quality, institutional memory, and justice visibility.
7. Provide multi-language examples in Python, R, Julia, SQL, Rust, Go, C, C++, and Fortran.
8. Export reproducible tables, figures, documentation, and notebook placeholders.

## Responsible-use note

The data are synthetic and the models are methodological examples. They are intended for learning, reproducible article companions, dashboard prototyping, and professional modeling design. Real slow-variable assessment requires empirical system data, local knowledge, field evidence, system history, governance context, uncertainty analysis, and public accountability.

Do not use these examples to rank communities, declare collapse risk, justify abandonment, or treat slow harm as inevitable without validated evidence and ethical review.

## Quick start

Run the dependency-light Python workflow:

```bash
python3 python/slow_variables_standard.py
```

Optional advanced Python workflow:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements-advanced.txt
python python/slow_variables_advanced.py
```

Run smoke checks:

```bash
bash smoke-test.sh
```
