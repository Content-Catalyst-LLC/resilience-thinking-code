# Modeling Principles

This companion scaffold models slow variables and hidden system change.

## Core variables

- maintenance backlog
- public trust
- ecological memory
- climate pressure
- adaptive capacity
- system memory
- monitoring quality
- justice visibility
- exposure
- threshold distance
- hidden risk
- fast shocks
- system function

## Core modeling logic

Slow variables shape resilience before fast events reveal vulnerability. Hidden risk rises when backlog, climate pressure, exposure, trust decline, memory loss, monitoring weakness, and justice invisibility accumulate. Threshold distance shrinks as buffers decline. Fast shocks do more damage when slow variables have already narrowed response space.

The scaffold calculates:

1. slow-variable profile scores
2. hidden-risk scores
3. threshold-distance estimates
4. fast-shock impacts under different slow-variable scenarios
5. adaptive-investment scenario comparisons
6. monitoring and justice visibility diagnostics
7. synthetic hidden-risk classifier outputs

## Interpretation

These examples are transparent modeling scaffolds, not empirical predictions. Real slow-variable assessment requires local data, system history, field evidence, community knowledge, domain expertise, uncertainty review, and accountable governance.
