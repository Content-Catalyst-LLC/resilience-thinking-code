# Responsible Use

This scaffold is not a collapse-prediction system.

Do not use it to:

- rank communities or institutions without validated evidence and public review
- treat slow harm as inevitable
- justify abandonment, displacement, austerity, or coercive intervention
- claim exact threshold distance from synthetic data
- ignore local knowledge, worker knowledge, Indigenous knowledge, resident reports, or frontline evidence
- shift responsibility from institutions to vulnerable communities
- replace accountable governance with model output

Use it to support transparent inquiry into how slow variables, hidden risk, adaptive capacity, memory, monitoring, and justice visibility shape resilience before crisis.
