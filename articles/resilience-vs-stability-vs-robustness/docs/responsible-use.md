# Responsible Use

This folder is designed to support predictive-model development for resilience analysis, but the included workflows use synthetic data.

Do use this scaffold for:

- methodological learning
- predictive pipeline design
- feature engineering
- scenario simulation
- reproducibility examples
- early dashboard prototyping
- model-governance planning

Do not use this scaffold for:

- real-world ranking of communities
- automated policy decisions
- insurance, lending, employment, or eligibility decisions
- justifying underinvestment in vulnerable populations
- replacing participatory planning or domain expertise

A resilience prediction should be treated as a structured warning signal, not a final judgment.
