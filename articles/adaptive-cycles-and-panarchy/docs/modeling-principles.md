# Modeling Principles

This companion scaffold models adaptive cycles and panarchy.

## Core variables

- potential
- connectedness
- resilience
- rigidity
- memory
- novelty
- disturbance exposure
- phase state
- release threshold
- revolt influence
- remember influence
- scale speed

## Core modeling logic

The adaptive cycle is represented as movement among four phases:

- `r`: growth or exploitation
- `K`: conservation
- `Omega`: release
- `alpha`: reorganization

A system becomes vulnerable to release when connectedness and rigidity rise while resilience falls. Reorganization depends on memory and novelty. Panarchy is represented through interacting faster and slower cycles: fast-cycle release can create revolt effects, while slow-cycle memory can support remember effects.

## Interpretation

These models are conceptual scaffolds, not empirical predictions. They help make adaptive-cycle logic explicit and reproducible, but real applications require historical evidence, ecological or institutional data, domain expertise, and ethical review.
