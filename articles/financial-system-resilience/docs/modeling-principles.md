# Modeling Principles

This companion scaffold models financial system resilience as a capital, liquidity, infrastructure, governance, inclusion, systemic-exposure, and implementation problem.

## Core variables

- capital strength
- liquidity resilience
- infrastructure robustness
- governance capacity
- inclusive resilience
- systemic exposure
- implementation burden
- leverage pressure
- liquidity fragility
- common asset exposure
- operational dependency
- climate financial exposure
- non-bank exposure
- household financial fragility
- policy coordination stress
- financial function under stress

## Core modeling logic

Financial resilience value rises when capital strength, liquidity resilience, infrastructure robustness, governance capacity, and inclusive resilience increase. Value falls when systemic exposure, implementation burden, credit losses, liquidity runs, fire sales, operational disruption, climate repricing, non-bank stress, household default pressure, and policy coordination stress increase.

The scaffold calculates strategy value, inclusion-adjusted value, infrastructure-adjusted value, dynamic financial function under repeated stress, implementation-burden diagnostics, systemic-exposure diagnostics, and Monte Carlo robustness under uncertainty.
